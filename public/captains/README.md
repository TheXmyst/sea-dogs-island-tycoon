# Dossier des images de capitaines

Placez l'image originale de Nadia ici avec le nom `nadia_original.png`

Ensuite, exécutez le script Python pour enlever le fond :
```bash
python remove_background.py nadia_original.png nadia.png 15
```

L'image traitée sera sauvegardée comme `nadia.png` et utilisée dans le jeu.
